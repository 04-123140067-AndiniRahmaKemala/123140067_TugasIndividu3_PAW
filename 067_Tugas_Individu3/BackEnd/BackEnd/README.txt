Product Review Analyzer Backend
===============================

A Pyramid-based backend for analyzing product reviews using AI.

Features:
- Sentiment analysis with Hugging Face
- Key points extraction with Gemini
- PostgreSQL database storage
- RESTful API endpoints

Setup:
1. Install PostgreSQL
2. Create database: product_review_db
3. Run: pip install -e .
4. Configure database in development.ini
5. Start server: pserve development.ini
